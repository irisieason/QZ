### 1. 通用工具创建

#### 🎣 `src/hooks/useControlledState.ts`
**功能**：统一处理受控/非受控组件模式

**包含方法**：
- `useControlledState<T>()` - 通用受控/非受控状态管理
- `useControlledBoolean()` - 布尔值状态管理（带 toggle 方法）

**优势**：
- ✅ 消除重复代码
- ✅ 统一行为逻辑
- ✅ 减少 bug 风险
- ✅ 便于测试和维护

**使用示例**：
```tsx
const [value, setValue] = useControlledState(
  props.value,
  props.defaultValue || '',
  props.onChange
);
```

#### 🎨 `src/utils/classnames.ts`
**功能**：类名组合和规范化工具

**包含方法**：
- `cn()` - 条件性组合类名（类似 classnames 库）
- `normalizeVariant()` - 规范化 Figma 变体名称
- `normalizeState()` - 规范化状态名称
- `createBEM()` - BEM 风格类名生成器

**优势**：
- ✅ 避免魔法字符串
- ✅ 统一类名生成逻辑
- ✅ 支持条件类名
- ✅ 轻量级（无外部依赖）

**使用示例**：
```tsx
const classes = cn(
  'button',
  `button--${normalizeVariant(variant)}`,
  isActive && 'button--active',
  className
);
```

### 3. 性能优化应用

#### useMemo 优化
```tsx
// 优化类名计算
const containerClasses = useMemo(() => cn(
  CLASSES.ROOT,
  `${CLASSES.ROOT}--${normalizeState(state)}`,
  isFocused && `${CLASSES.ROOT}--${STATE_MODIFIERS.FOCUSED}`,
  className
), [state, isFocused, className]);

// 优化属性对象
const inputProps = useMemo(() => ({
  ref, id, type: 'text' as const,
  // ...
}), [ref, id, inputValue, state, ...]);
```

#### useCallback 优化
```tsx
// 所有事件处理器都使用 useCallback
const handleChange = useCallback((event) => {
  const newValue = event.target.value;
  setInputValue(newValue);
  onChangeRef.current?.(newValue, event);
}, [setInputValue]);
```

#### useRef 优化
```tsx
// 使用 ref 存储回调，避免依赖变化导致重新创建函数
const onChangeRef = useRef(onChange);
useEffect(() => {
  onChangeRef.current = onChange;
}, [onChange]);
```
